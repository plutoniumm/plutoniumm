# Atomicss Usage
Atomicss is a CSS library that provides a set of utility classes to help you build your website faster. It is
minimal as shit and lightweight on minification

**Note:**
- I will GENERALLY not change this
- No old classes will be changed
- I may add new styles anytime

## Add
```html
<link rel="stylesheet" href="//manav.ch/atomic.css">
```

## Margin, Padding & Border Radius
### Individual
Allowed Values `0, 2px, 5px, 10px, 20px`

- **.mA**: margin: Apx;
- **.pA**: padding: Apx;
- **.rxA**: border-radius: Apx;

```html
<div class="m0 p2 rx5">
  margin: 0;
  padding: 2px;
  border-radius: 5px;
</div>

<div class="m10 p20">
  margin: 10px;
  padding: 20px;
</div>
```

### Combination
Allowed Values `5px, 10px, 20px`

- **.rpm-A**: margin: Apx; padding: Apx; border-radius: Apx;

```html
<div class="rpm-5">
  margin: 5px;
  padding: 5px;
  border-radius: 5px;
</div>
```

## Text Behaviour

- **.fwA**: font-weight: A00; (Allow from 100-700)
- **.tA**: text-align: `l | r | j | c` (left | right | justify | center)
- **.j-A**: justify-content `ct | bw | ar` (center | space-between | space-around)

```html
<div class="fw3 tr j-ar">
  font-weight: 300;
  text-align: right;
  justify-content: space-around;
</div>

<div class="fw7 tj j-ct">
  font-weight: 700;
  text-align: justify;
  justify-content: center;
</div>
```

## Layouts
- **.w-A**: width: auto | A%; (Allowed `auto, 100, 50, 33, 25, 20, 0`)
- **.h-A**: height: auto | 100vh | A% ; (Allowed `auto, 1vh, 100, 50, 33, 25, 20, 0`)
- **.d-A**: display: `n | b | ib | if | u` (none | block | inline-block | inline-flex | unset)
- **.p-A**: position: `abs | rel | fix | stx` (absolute | relative | fixed | sticky)

```html
<div class="w-100 h-1vh d-b p-abs">
  width: 100%;
  height: 100vh;
  display: block;
  position: absolute;
</div>

<div class="w-50 h-50 d-ib p-rel">
  width: 50%;
  height: 50%;
  display: inline-block;
  position: relative;
</div>
```

## Minor Specifics
- **.o-A**: opacity: A%; (Allowed `0, 25, 33, 50, 66, 75, 100`)
- **.b0**: border: 0 !important; outline: none !important;
- **.flow-D-A**: overflow-D: `hidden | scroll` (D = `x | y`)
- **.ptr**: cursor: pointer;
- **.mx-a**: margin: 0 auto;

```html
<div class="o-50 b0 flow-x-hidden ptr mx-a">
  opacity: 50%;
  border: 0 !important;
  outline: none !important;
  overflow-x: hidden;
  cursor: pointer;
  margin: 0 auto;
</div>

<div class="o-33 flow-y-scroll">
  opacity: 33%;
  overflow-y: scroll;
</div>
```
