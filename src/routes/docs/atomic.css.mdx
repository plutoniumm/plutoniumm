# Atomicss Usage
Atomicss is a CSS library that provides a set of utility classes to help you build your website faster. It is
minimal as shit and lightweight on minification

## Margin, Padding & Border Radius
### Individual
Allowed Values `0, 2px, 5px, 10px, 20px`

- **.mA**: margin: Apx;
- **.pA**: padding: Apx;
- **.rxA**: border-radius: Apx;

```html
<div class="m0 p2 rx5">
  margin: 0;
  padding: 2px;
  border-radius: 5px;
</div>

<div class="m10 p20">
  margin: 10px;
  padding: 20px;
</div>
```

### Combination
Allowed Values `5px, 10px, 20px`

- **.rpm-A**: margin: Apx; padding: Apx; border-radius: Apx;

```html
<div class="rpm-5">
  margin: 5px;
  padding: 5px;
  border-radius: 5px;
</div>
```
