Zero, One, Two

Over the last 60 years.

  • Compilers did not make programming go away, it just changed how we do it, with higher level languages
  • Dreamweaver did not make programming go away, it just showed us a better way of seeing it
  • Wordpress did not make programming go away, it just made it faster for some standardised use cases
  • No-Code did not make programming go away it in fact got more poeple into mobile dev
  • LSP did not automate away code generation it just made it faster to check for errors, write specifications and documentation
  • Quantum Computing will not make programming go away, it will just speed up like some integer % of the use cases
  • AI will not make programming go away, it will just make it harder to get into.

This DOES NOT mean, that programming will end. It will just get harder.
